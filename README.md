MicrostructureAlpha is a high-frequency trading system that predicts short-term price moves using Hawkes Buy-Sell Imbalance (BSI) and multinomial logistic regression or MLP classification. It ingests raw TAQ trade data, filters and resamples it into bars, then computes BSI to capture order flow pressure alongside features like short-term returns, volatility, and VWAP deviation. Signals are generated when predicted probabilities exceed a confidence threshold, opening long or short positions that are held until BSI crosses zero — indicating an order flow reversal. Execution can be simulated realistically with latency, slippage, partial fills, and borrowing costs, with each trade's entry, exit, duration, and PnL recorded. 

NYSE TAQ data is not included as it is proprietary and licensed by the exchange, users must provide their own data with columns DATE, TIME_M, PRICE, SIZE, TR_CORR, and TR_SCOND.
